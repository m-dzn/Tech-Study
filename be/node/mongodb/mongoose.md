# Mongoose

> ODM (Object Document Mapper)

## 특징

1. Schema 존재
2. JOIN과 유사한 populate 기능 지원

## 스키마

### 자료형

| 종류     | 설명            |
| -------- | --------------- |
| String   | 문자열          |
| Number   | 숫자            |
| Date     | 날짜            |
| Buffer   | 바이너리 데이터 |
| Boolean  | 논리            |
| Mixed    | 혼합 타입       |
| ObjectId | 식별자          |
| Array    | 배열            |
