# Mongo DB 조회

## 검색 조건

| 종류               | 코드                                  |
| ------------------ | ------------------------------------- |
| 모든 다큐먼트 조회 | .find({ })                            |
| name 필드 조회     | .find({ }, {  _id: 0, name: 1, ... }) |
| 초과               | $gt                                   |
| 이상               | $gte                                  |
| 이하               | $lte                                  |
| 미만               | $lt                                   |
| 같지 않음          | $ne                                   |
| 또는               | $or                                   |
| 포함               | $in                                   |
| 불포함             | $nin                                  |
| 부정               | $not                                  |



## 정렬

| 종류          | 코드       |
| ------------- | ---------- |
| 정렬          | .sort({ }) |
| 개수 제한     | .limit( )  |
| 개수 뛰어넘기 | .skip( )   |
