# FormData



## 메서드

| 메서드          | 기능                  |
| ------------ | ------------------- |
| append(키, 값) | 키에 값을 추가합니다         |
| set(키, 값)    | 키의 값을 새로 세팅합니다      |
| get(키)       | 키에 저장된 값 하나를 가져옵니다  |
| getAll(키)    | 키에 저장된 모든 값을 가져옵니다  |
| has(키)       | 키에 저장된 값이 있는지 체크합니다 |
| delete(키)    | 키에 저장된 값을 모두 삭제합니다  |
