# JS

## 파일 분리의 필요성

JS 파일을 분리해야 하는 이유는 다음과 같습니다.

브라우저는 .js 확장자를 가진 파일을 캐싱하므로 성능 상 유리합니다.

HTML 마크업 언어와 JS 스크립트가 분리되면서 가독성 또한 높아집니다.



※ src 속성이 있으면 <script> 태그 안에 적힌 코드는 무시됩니다.
